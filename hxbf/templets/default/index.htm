活动形式 Situation

  1. 按县、市、省选出优秀代表。
  2. 从家庭民族贡献、传统文化、社会义务、美貌、口才、形象等多维度评比。
  3. 省选出各民族代表参加国家级选拔。
  4. 国家级选拔采用异地选拔方式，在本民族最具代表性的地区选拔。
  5. 选出56个民族前三名华夏夫人代表。
  6. 带着中华民族传统文化，沿着丝绸之路到各国进行文化、才艺、服饰展览

活动成果 Activity results

一、大赛组委会

  1. 成立华夏夫人研究院。旨在开展传统文化教育传承，形象口才培训，家庭教育培训，社会慈善活动等。
  2. 成立深圳市华夏夫人品牌发展有限公司（经纪公司），运营华夏夫人品牌及各民族形象大使。
  3. 为拍摄56个民族大电影找到题材，向建党100周年献礼。
  4. 为56个民族纪录片《鼓舞中国》拍摄找到素材，记录、传播、传承各民族优秀的传统文化。

二、参选夫人

  1. 成为华夏夫人家庭委员会股东，享受盈利分红。
  2. 相识一群志同道合的姐妹，共同传承、发展家庭教育和民族传统文化。
  3. 获得本民族传统文化，民族舞蹈、服饰、礼仪等的系统课程。
  4. 获得价值三万元的个人介绍短片、珠宝、服饰等产品。
  5. 能够代表国家，代表本民族出国交流，把民族传统文化传播到世界各地。
  6. 参演本民族的大电影和纪录片拍摄。
  7. 签约深圳市华夏夫人品牌发展有限公司以及经纪公司，成为华夏夫人旗下的签约艺人。

三、社会影响

  1. 响应“一带一路”战略，促进文化产业通道建设；以文明交流超越文明隔阂，文明互鉴超越文明冲突，打造中国外交新图景。
  2. 满足丝绸之路沿线国家居民的文化消费需求，进而增进相互的亲近感和文化认同。
  3. 促进文化产品、文化要素跨国跨地区的自由流动和中国各地旅游产业的发展。
  4. 促进国人对中华各民族传统文化发掘、保护和传承；积极培育和践行社会主义核心价值观。
  5. 形成了以华夏夫人为品牌，以传承中华各民族传统文化为使命的文化产业。
